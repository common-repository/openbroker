== Description ==
* Author: [teamopenbroker](https://openbroker.com)
* Project URI: <https://www.openbroker.com>

This is official OpenBroker.com plugin which enables WordPress users to build Real Estate websites with no programming knowledge.

= Tags =
OpenBroker, Real Estate Website, Real Estate MLS, MLS Website, Real Estate API, Real Estate CRM

== About ==

This plugin is designed for all Real Estate Agents who need a website, and WordPress developers seeking to create Real Estate Websites for their clients. For now, we are only available in Spain, Portugal, Italy, and France. The installation process is very simple, and our support team is always here to assist you in any regard. This plugin connects your WordPress to OpenBroker’s Listing’s database, so that you can display the entire MLS on your Website, in real time, at speed-of-light, SEO-Ready, via our API.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly;
2. Activate the plugin through the 'Plugins' screen in WordPress

= How to use this plugin? =
1. Open your plugin settings page in WordPress Dashboard and enter your API credentials (can be found at your OpenBroker dashboard)
2. Build your pages using shortcodes

== How it works ==

1- Create an account on www.openbroker.com
2. Install the plugin
3. Create your website using the instructions provided at installation step
4. Once your website is done, obtain your token on www.openbroker.com, under “My Website” section
5. Upgrade your plan to “Basic”, “Broker”, or “Enterprise”. Plans start as cheap as 29€/month and provide limitless functionality for your website.
